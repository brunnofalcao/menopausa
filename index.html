Perguntas frequentes

Dúvidas comuns.

Este guia substitui consulta médica? +

Não. Este guia tem caráter educativo e informativo. O objetivo é ajudar você a entender o que está acontecendo no seu corpo e conhecer as alternativas disponíveis — mas o acompanhamento médico individualizado é insubstituível.

O conteúdo serve para quem ainda não iniciou a menopausa? +

Sim. O guia é útil tanto para quem está na perimenopausa (transição) quanto para quem já passou pela menopausa. Entender o processo com antecedência torna as decisões mais conscientes e os sintomas menos assustadores.

Preciso ter uma contraindicação para usar hormônios para aproveitar o guia? +

Não necessariamente. Muitas mulheres que poderiam usar hormônios simplesmente preferem explorar alternativas primeiro. O guia é para qualquer mulher que queira entender o que acontece no seu corpo e ter mais controle sobre as próprias escolhas.
